Command to Set local Configuration to use git in local
======================================================

1. git config --global user.name "********"
2. git config --global user.email "**********"

Command to clone git repository to local repository(run the command on respective folder)
===========================================================

1. git clone (link/origin)

Command to add,commit and push file to github
==============================================

1. add file in git repository(staging area)- git add filename
2. To commit file - git commit -m "message"
3. push file to github - git push origin "main(folder name)"

Command to move new folder to gitup

1.inside folder run - git init
2.To connect new folder to github repository - git remote add origin "link"
How to Create new branch ,view branch,change branch 1. To list branch command - git branch -a 2. create new branch in gitup and use - (git pull) to pull change from github to local folder 3. change branch - git checkout [branch name].
